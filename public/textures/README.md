# 3D Seamless Textures

This folder contains seamless, tileable textures for 3D rendering.
These are DIFFERENT from swatch images in /stones/ folder.

## Structure
- `/stones/` - Seamless tileable stone textures (2048x2048 minimum)

## Naming Convention
- `{stone-id}.jpg` - Main diffuse/color texture
- `{stone-id}-normal.jpg` - Normal map (optional)
- `{stone-id}-roughness.jpg` - Roughness map (optional)

## Important
- These textures must be seamless/tileable
- Never use PDF grids, collages, or lifestyle images
- Minimum resolution: 2048x2048
